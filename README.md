AI Chatbot
A modern, interactive chatbot application built with React and Node.js that provides real-time responses to user queries. Features a clean, responsive interface with smooth animations and persistent conversation history.
Features
Real-time messaging interface
Contextual response system
Message persistence using SQLite
Responsive design with Tailwind CSS
User-friendly UI with message timestamps and typing indicators

Tech Stack

Frontend: React + Vite
Backend: Node.js + Express
Database: SQLite
Styling: Tailwind CSS
Icons: Lucide React
